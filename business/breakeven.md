# breakeven.md

## Unit Economics & Break-even Analysis

### Cost per Active User
- **Compute Costs**: $0.10/user/month (based on estimated server usage for AI processing)
- **Storage Costs**: $0.05/user/month (average metadata storage requirement)
- **Bandwidth Costs**: $0.02/user/month (average data transfer for metadata updates)
  
**Total Cost per Active User**:  
**$0.10 + $0.05 + $0.02 = $0.17/user/month**

### Pricing Tiers
| Tier Name       | Price ($/mo) | Features                                                                 |
|------------------|--------------|--------------------------------------------------------------------------|
| Basic            | $5           | Metadata correction for up to 100 titles, basic support                 |
| Pro              | $15          | Metadata correction for up to 500 titles, priority support, reporting    |
| Premium          | $30          | Unlimited titles, advanced AI features, 24/7 support, analytics dashboard |

### Customer Acquisition Cost (CAC) Range
- **CAC Estimate**: $25 - $50 per user (includes marketing, sales, and onboarding costs)

### Lifetime Value (LTV) Estimate
- **Average Revenue per User (ARPU)**:  
  - Basic: $5/month  
  - Pro: $15/month  
  - Premium: $30/month  
- **Average Customer Lifespan**: 24 months  
- **LTV Calculation**:  
  - Basic: $5 * 24 = $120  
  - Pro: $15 * 24 = $360  
  - Premium: $30 * 24 = $720  

### Break-even Users Count
- **Break-even Point Calculation**:  
  - Fixed Costs (assumed): $2,000/month  
  - Contribution Margin per User: Price - Cost per User  
    - Basic: $5 - $0.17 = $4.83  
    - Pro: $15 - $0.17 = $14.83  
    - Premium: $30 - $0.17 = $29.83  
- **Break-even Users Count**:  
  - Basic: $2,000 / $4.83 ≈ 414 users  
  - Pro: $2,000 / $14.83 ≈ 135 users  
  - Premium: $2,000 / $29.83 ≈ 67 users  

### Path to $10K MRR
- **Target Monthly Recurring Revenue (MRR)**: $10,000  
- **Required Users Count**:  
  - Basic: $10,000 / $5 = 2,000 users  
  - Pro: $10,000 / $15 = 667 users  
  - Premium: $10,000 / $30 = 334 users  

**Recommended Path**:  
- Focus on the **Pro Tier** with **667 users** to achieve $10K MRR, balancing features and affordability.
